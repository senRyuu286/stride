A lightning-fast, local-first academic command center designed to prevent student burnout. Stride unifies your academic life by categorizing tasks, quizzes, and exams by urgency, running entirely in your browser without the need for an external database.

**Note:** Stride strictly utilizes `LocalStorage` to function effectively without latency. Check out the [Live Demo here](https://senryuu286.github.io/stride).

## Features

- **The Inbox (Today):** Low-friction, immediate daily tasks.
- **The Radar (Upcoming):** Quizzes and assignments due within a rolling 7-day window.
- **The Horizon (Future):** Major exams and midterms with visual progress countdowns.
- **Local-First Architecture:** Data is persisted instantly to the browser's storage. Works perfectly offline.
- **Responsive Design:** Gracefully collapses from a desktop grid into a mobile-friendly tabbed view.
